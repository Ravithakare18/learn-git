Git Initialise >>

Now its time to Initialise git :)

> Initialise git
    git init

> Check status
    git status

> Track the files
    git add <filename> or
    git add . (for all the untrack files)

> Check status either files added to track or not
    git status

> Commit all the files with appropriate message
    git commit -m '<your comments>'

> Check status and log
    git log

Well Done. Good job :)
